this is a prototype autoiplot command program for basic movements for an autonomous drone. 
This program is created on python programming lanaguage by using pymavlink library.
pymavlink is a general library for mavlink protocol.
it is primarily used to implement MAVlink protocol to communicate in many Mavlink based systems such as developer API(Dronekit) and other applications.
you need atleast python 3.5 or upgrade to excecute the code.
